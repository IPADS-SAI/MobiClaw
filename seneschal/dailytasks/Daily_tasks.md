# MobiAgent 每日活动信息采集任务集（v1）

> 本文档定义了一组可直接下发给 **MobiAgent（手机 GUI-Agent）** 的标准化任务，用于在用户授权前提下，自动收集其每日数字活动信息，作为「个人助理 Agent / 数据管家」的输入数据源。

---

## 设计目标

* 可在 **真实手机环境** 中稳定执行
* 不依赖用户实时交互
* 以 **结构化摘要** 为输出，而非原始隐私内容
* 支持长期、低干扰的自动巡检
* 为后端 Agent 的分析、提醒、规划提供高质量信号

---

## 一、📦 电商 / 消费类任务

### Task 1：查看淘宝当前待收货订单

**任务描述**

```
打开淘宝 App，
进入「我的淘宝」→「待收货」页面，
依次查看当前所有待收货订单，
对每个订单提取关键状态信息后返回。
```

**关注信息**

* 商品名称（简要）
* 订单状态（已发货 / 派送中 / 即将签收）
* 预计送达时间（如有）
* 是否存在异常提示（延误、地址异常等）

**期望输出**

```json
{
  "app": "Taobao",
  "task": "pending_delivery_check",
  "orders": [
    {
      "item": "蓝牙键盘",
      "status": "运输中",
      "eta": "明日送达",
      "risk": false
    }
  ]
}
```

---

### Task 2：查看淘宝待付款 / 即将关闭订单（可选）

**任务描述**

```
打开淘宝 → 我的淘宝 → 待付款，
检查是否存在即将关闭的订单，
如有，记录商品与剩余时间。
```

## 二、💬 社交 / 沟通类任务

### Task 3：查看微信聊天列表前 5 个联系人最新消息

**任务描述**

```
打开微信 App，
进入聊天列表，
依次查看聊天列表中前 5 个联系人，
不进入聊天详情页，
提取每个联系人的最新一条消息摘要与时间。
```

**注意事项**

* 不滚动历史记录
* 不读取完整对话内容
* 仅使用聊天列表可见信息

**关注信息**

* 联系人名称
* 最新消息类型（文字 / 语音 / 图片 / 系统提示）
* 最新消息时间
* 未读消息数量

**期望输出**

```json
{
  "app": "WeChat",
  "task": "recent_chat_overview",
  "chats": [
    {
      "contact": "张三",
      "last_message_type": "text",
      "time": "10:42",
      "unread_count": 2
    }
  ]
}
```

---

### Task 4：标记需要跟进的聊天（辅助任务）

**任务目标**

* 为后端 Agent 提供“可能需要用户后续行动”的信号

**典型标记条件（示例）**

* 连续多条未读消息
* 最新一条消息为疑问句
* 出现关键词：

  * “可以吗”
  * “麻烦你”
  * “尽快回复”

> 本任务仅做 **信号标记**，不做价值判断

## 三、📱 手机整体使用行为任务

### Task 5：查看今日手机 App 使用情况

**任务描述**

```
打开系统设置，
进入「屏幕使用时间 / 应用使用情况」，
切换至“今日”视图，
提取使用时长排名前 10 的应用。
```

**关注信息**

* App 名称
* 使用时长
* 使用次数（如可见）
* 是否存在明显异常使用

**期望输出**

```json
{
  "app": "System",
  "task": "daily_app_usage",
  "top_apps": [
    {
      "name": "WeChat",
      "duration_minutes": 86
    }
  ]
}
```

---

### Task 6：检测异常使用模式（辅助）

**示例场景**

* 深夜长时间使用娱乐类 App
* 工作时段娱乐类 App 占比过高

> 仅标记异常模式，不输出评价或结论

## 四、📅 日程 / 待办类任务

### Task 7：查看系统日历今日及未来 3 天事项

**任务描述**

```
打开系统日历，
查看今日、明日及未来 3 天的日程安排，
提取事件名称、时间、是否为全天事件。
```

---

### Task 8：查看系统待办 / 备忘录未完成事项

**任务描述**

```
打开系统待办事项或备忘录 App，
查看未完成事项列表，
提取事项名称与截止时间（如有）。
```

---

## 五、📩 邮件 / 工作沟通类任务（可选）

### Task 9：查看邮箱未读邮件摘要

**任务描述**

```
打开邮箱 App，
查看未读邮件列表，
提取前 5 封未读邮件的发件人、主题与时间。
```

---

## 六、📸 截图 / 文件变化监测任务（进阶）

### Task 10：检测相册中今日新增截图

**任务描述**

```
打开相册 App，
进入「截图」分类，
查看今日是否新增截图，
如有，记录数量与生成时间。
```

**说明**

* 不解析图片内容
* 图片语义分析交由后端 Agent 处理
